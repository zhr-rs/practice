# 📈 新闻文本词频分析系统
基于 Streamlit 开发的轻量化新闻文本分析工具，支持**网页内容爬取**、**中文分词**、**词频统计**、**多维度可视化**（8种图表），开箱即用。

## ✨ 核心功能
- 🕷️ 网页内容爬取：支持多链接批量爬取，自动过滤无关标签（script/style等）；
- 🔤 中文分词+过滤：集成 jieba 分词，支持停用词/低频词过滤；
- 📊 多维度可视化：提供柱状图/横向柱状图/折线图/饼图/面积图/热力图/词云图/美化表格共8种展示方式；
- 🎨 交互优化：Streamlit 侧边栏配置，支持词频阈值动态调整，结果实时渲染；
- 💾 本地保存：爬取内容自动保存为 txt 文件，方便后续复用。
